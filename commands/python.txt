Check PIP version:
	$ pip --version

Download a package named "camelcase":
	$ pip install camelcase

Uninstall the package named "camelcase":
	$ pip uninstall camelcase

List installed packages:
	$ pip list
	OR
	$ pip list -v

To get a list of installed packages and versions
  $ pip freeze > [output filename]
  Example:
  $ pip freeze > requirements.txt

To install packages mentioned in requirement.txt
  $ pip install -r requirements.txt

To uninstall packages mentioned in requirement.txt
  $ pip uninstall requirements.txt

  $ python -m venv .venv

START:Download Youtube video using `yt-dlp` Python package.
===========================================================

Install package
	$ pip install yt-dlp

Download subtitles
	$ yt-dlp --write-subs --sub-lang ur --skip-download "youtube video link"

Download Youtube video
	$ yt-dlp "https://www.youtube.com/watch?v=uSuNPCzA4SQ"

Download Audio only
	$ yt-dlp -f "bestaudio" --extract-audio --audio-format mp3 "https://www.youtube.com/watch?v=uSuNPCzA4SQ"
